Atom Project



atom project is an open source set of elements and builders, aimed at making web app development fast and simple.

atom has a set of elements which are web components built with polymer. There are elements for building simple forms, displaying structured data (like tables & lists), and creating typical web pages.

atom Elements

A collection of basic and composite form elements, data display elements like tables and lists, and simple page elements like headers, footers, loaders, notifications, maps, tabs and more. These are basic building blocks that can be used to create web and mobile applications.

travelnxt Elements

A collection of web components for creating niche travel experiences built by Tavisca. Elements include everything you need to create search to book travel flows for different products like flights, hotels or cars. This featured component set is built on top of atom elements.

Sample App

We have created a sample application for a hotel search to book flow using atom and travelnxt elements. It is open source and available for you to tryout.

Tools

Interactive Documentation

The atom elements browser is an interactive documentation system for web components. For any web component, it automatically generates a panel using which the user can interact with the public properties of the element. Support for interacting with methods and events of web components will added later. This documentation portal can be set up simply using a CLI.

  • More

Form Builder

Use the form builder to create amazing forms in seconds by dragging and dropping elements. The form builder is designed to create forms used in web applications, not surveys. Customise all important aspects of forms like labels, inputs, actions, error messages and more.

  • More

Page Builder

Provides a simple mechanism for creating responsive layouts using web components. Use our elements to get started or add any polymer element you want. You can also preview, download or share your designs without the need to sign up.
