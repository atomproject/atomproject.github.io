About

Atom project is an open source set of elements and builders, aimed at making we app development fast and simple.
Atom has a set of elements which are web components built with polymer. These are elements for building simple forms, displaying structured data (like tables & lists), and creating typical web pages.
Atom also has a set of tools that ease the app creation interactive documentation, form builder and page builder.

Atom Elements

A collection of basic and composite form elements, data display elements like tables and list, and simple page elements like headers, footers, loaders, notifications, maps, tabs and more. There are basic building blocks that can be used to create web and mobile applications.

Travelnxt Elements

A collection of web components for creating niche travel experiences built by Tavisca. Elements include everything you need to create search to book travel flows for different products like flights, hotels or cars.

Interactive Documentation

The atom elements browser is an interactive documentation system for web components. For any web components, it automatically generates a panel using which the user can interact with the public properties of the elements. Support for interacting with methods and events will added later. This documentation portal can be set up simply using a CLI.

Form Builder

Use the form builder to create amazing forms in seconds by dragging and dropping elements. The form builder is designed to create forms used in web applications, not surveys. Customise all important aspects of forms like labels, inputs, actions, error messages and more.

  • More

Page Builder

Provides a simple mechanism for creating responsive layouts using web components. Use our elements to get started with or add any polymer element you want. You can also preview, download or share your designs without the need to sign up.
